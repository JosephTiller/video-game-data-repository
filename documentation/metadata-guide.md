Metadata fields used in this repository:

- game: Name of the game
- release_year: Year the game released
- copies_sold: Number of copies sold
- developer: Company that made the game
- mob: Name of the Minecraft mob
- type: Category or class
- hostile: Whether the mob attacks players
- health: Health points
- weapon: Name of the Fortnite weapon
- rarity: Weapon rarity
- damage: Damage dealt
- season: Fortnite season name
- chapter: Fortnite chapter number
- start_year: Year the season started
- theme: Main theme of the season
